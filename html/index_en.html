OAG (Orthogonal Array Generator)

Japanese page

abstract

OAG (Orthogonal Array Generator) is very simple test case generator using an orthogonal or nearly-orthogonal arrays.

  • command line tool
  • PICT style model file
  • Free

  • construction strategy for Orthogonal Array

    1. if all factors have same level and prime number, then use EX-OR extension.
    2. if the model file has a mixed level or EX-OR extension is failed,
      then try Hongquan Xu's algorithm (*1) .

  • and, some other algorithms can be enabled.

    1. Pairwise ( using qict routine)
    2. All Combinations
    3. Random

  • this tool can check an array ; orthogonal array or nearly-orthogonal array.

(*1)
http://www.stat.ucla.edu/~hqxu/pub/Xu2002.pdf

usage

generate orthogonal array

OAG   model-file-name

check array

OAG   -check   array-file-name

model-file format

ParamName1 : Value1-1, Value1-2, Value1-3, ...
ParamName2 : Value2-1, Value2-2, Value2-3, ...

 ( delimiter is comma )

array-file-name format

ParamName1  ParamName2 ...
Value1-n    Value2-n ...
Value1-n    Value2-n ...
Value1-n    Value2-n ...

 ( delimiter is tab )


example

generate orthogonal array

model_file_01 

AAA : a_1,a_2,a_3,a_4
BBB : b_1,b_2
CCC : c_1,c_2
